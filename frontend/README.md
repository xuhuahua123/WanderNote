# frontend

移动端 H5 前端。

## 技术栈

计划使用：

```text
Vue 3
Vite
TypeScript
Vant UI
Pinia
Axios
```

## 主要职责

- 访问码登录页
- 首页 / 路线列表
- 路线详情页
- 添加城市 / 添加地点
- 照片选择弹窗
- AI 写作准备页
- 文章结果页
- 同步状态页
